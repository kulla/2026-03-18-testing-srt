# Repo for testing [@anthropic/sandbox-runtime](https://www.npmjs.com/package/@anthropic-ai/sandbox-runtime)

## Setup

1. Clone repo and run `npm install`.
2. Go into one of the directories and run the scripts there.

## Notes

For demonstrating some bugs this project patches `@anthropic/sandbox-runtime` by showing the wraped sandboxed command (see [patches](./patches)).
